# Sensei (HackMIT 2015)

###### Utilization of Kalman Filters to detect and analyze motion of people for real-world analytics.

[![Sensei in action](https://j.gifs.com/m2GGMJ.gif)](https://www.youtube.com/watch?v=bkN14S4BR5U)
